## How to run
- Clone the repo and open its directory
```
git clone https://github.com/El-Nebo/Udemy-Replica.git
cd .\Udemy-Replica\
```

- Download node_modules 
```
yarn
```

- Run and watch the project 
```
yarn run run
```
